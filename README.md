# Restaurant Demand Prediction

This project predicts restaurant food demand using a Machine Learning model.

## Tech Used
Python
Pandas
Scikit-learn
RandomForestRegressor

## Files
train_model.py - trains the ML model  
predict.py - loads the model and predicts demand  
restaurant_demand_model.pkl - trained model  
data/sample_input.csv - example input data  

## Run the project

Install libraries

pip install -r requirements.txt

Run prediction

python predict.py
